Go&Study — Обучение
Платформа использует Supabase для хранения прогресса. Нужно подключить один раз.
1
Создай Supabase проект
Зайди на supabase.com, создай новый проект. Запомни пароль от БД — он понадобится.
2
Запусти SQL миграцию
В Supabase → SQL Editor → вставь и выполни этот код:
SQL
-- Таблица прогресса студентов CREATE TABLE IF NOT EXISTS student_progress ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, student_login TEXT NOT NULL, step_key TEXT NOT NULL, completed BOOLEAN DEFAULT false, completed_at TIMESTAMPTZ, UNIQUE(student_login, step_key) ); -- Таблица данных (студент → Павел, Павел → студент) CREATE TABLE IF NOT EXISTS student_data ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, student_login TEXT NOT NULL, field_key TEXT NOT NULL, field_value TEXT NOT NULL, direction TEXT NOT NULL CHECK (direction IN ('from_student', 'from_pavel')), updated_at TIMESTAMPTZ DEFAULT NOW(), UNIQUE(student_login, field_key, direction) ); -- RLS: отключаем (используем anon key — доступ через JS) ALTER TABLE student_progress DISABLE ROW LEVEL SECURITY; ALTER TABLE student_data DISABLE ROW LEVEL SECURITY;
3
Вставь ключи из Supabase
Settings → API → Project URL и anon public key.
⚡ Ключи сохранятся в localStorage этого браузера. Нужно настроить один раз на каждом устройстве.
